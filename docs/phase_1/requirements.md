# System Requirements

## Overview

This document defines all high-level requirements for the KAFA Automated WhatsApp Certificate System.

---

## Company Information (Static)

- Company Name
- Registration Number
- Logo
- Address (Siège Social)
- Phone
- Email
- Website
- Authorized Signatories
- Official Seal

---

## Member Information (Dynamic)

- Full Name
- Date of Birth
- Identification Number
- Identification Type
- Address
- Member Number
- Date of Membership
- WhatsApp Phone Number (Required)

> A valid WhatsApp-enabled phone number is mandatory.

---

## Certificate Requirements

- Inject dynamic data into official template
- Preserve formatting
- Automatically insert issue date
- Generate PDF version
- Generate JPEG version

---

## Delivery Requirements

- Send certificate via WhatsApp
- Attach PDF OR secure S3 link
- Store delivery status in database
